* Installation

1. Clone this repo
2. Copy =nepes-dark.tmTheme= and =nepes-light.tmTheme= to =~/.config/bat/themes/=
3. Run =bat cache --build=
4. Set theme: =export BAT_THEME="Nepes Dark"=

* Configuration

#+begin_src shell
export BAT_THEME="Nepes Dark"  # or "Nepes Light"
#+end_src
